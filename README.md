# TryHackMe Write-ups

This repository contains my write-ups for rooms completed on TryHackMe, focusing on practical cybersecurity skills such as:

- Enumeration techniques
- Privilege escalation

Each write-up documents my methodology, tools used, and reasoning throughout the process.
